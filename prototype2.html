Planned Interactivity

I have planned for updating the d3 visualizations with following interactivities:

  • Highlighting all the circles related to a particular type if one of the circle of that type is hovered on i.e. Fade all of the non-hovered circles belonging to other types.
  • Providing a dropdown to filter data on different months.
  • on demand data display i.e. when a user hovers on a circle then a tool tip pops up with specific details of the data

How this visualization supports the theme?

Visualization helps in analysing the most active/busiest station and batallion and what type of incidents do they receive the most.
